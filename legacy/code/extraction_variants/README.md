# extraction_variants

Estatus: `legacy`

Aqui viven variantes historicas de extractores por persona, caso o criterio puntual.

Se conservan porque:

- documentan decisiones o pruebas previas
- pueden servir para auditoria o comparacion historica

No deben usarse como base estructural del flujo canonico.

El flujo activo de extraccion vive en `src/extraction/`.
